PARLAMENTO2030.ES
=============

## Requirements

NodeJS > 10.0

## Setup

```
git clone https://github.com/politicalwatch/parlamento2030.es.git
cd parlamento2030.es
npm install
cp src/config/index.js.example src/config/index.js
```

Finally, edit *config/index.js* file with your specific values.


## Run

```
npm run serve
```
